# Volleyball Coach Management System

Flask приложение за управление на волейболен отбор.

## Функционалности
- Управление на състезатели
- Проследяване на плащания
- Изпращане на напомняния

## Инсталация
1. Клонирайте репозитория:
   ```bash
   git clone https://github.com/anatoli9010/VolleyballCoach.git
   ```
2. Инсталирайте зависимости:
   ```bash
   pip install -r requirements.txt
   ```
